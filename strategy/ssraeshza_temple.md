Ssraeshza Temple can be confusing to navigate for those unfamiliar with the zone. There are multiple floors that are connected to each other via tubes. Below you can see how to get to each boss.

## Second Floor

You'll notice there are two locked doors marked Zazush Key and Zherozsh Key. These keys are uncommon drops off Commander Zazush and Commander Zherozh, respectively. These doors cannot be Pick Locked otherwise.

![ssra-floor-2](/assets/img/ssra-floor-2.jpg)

**Bosses on Second Floor:**

- **[Rhag'Mozdezh (Rhag 2)](/strategy/rhag`mozdezh)** - Located in the bottom right area, accessible via the second floor.
